# 📚 Daily Knowledge App

ความรู้ใหม่ทุกวันสำหรับพี่เบส

## 🔗 เปิดใช้งาน

1. ไปที่ GitHub Pages settings
2. เปิด Source: `main` branch
3. เปิดใช้งาน

## 📱 Features

- 📂 หมวดหมู่: OpenClaw, AI, Tech, Trading
- 🔥 บทความล่าสุด
- 🔖 บันทึกบทความ
- ⚙️ ตั้งค่า

## 🚀 Deploy

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/username/daily-knowledge.git
git push -u origin main
```

Then enable GitHub Pages in repository settings.
